# Home

## Hero

Valymux is the simple gateway for developers building with AI providers.

One place to connect, route, translate, and observe LLM traffic without fighting every provider’s API shape, rules, and quirks.

## Subheadline

We are building the layer AI engineers wish they had from day one: safe by default, fast under load, and transparent enough to trust in production.

## Primary Message

Every week brings a new model. Every provider brings a new format.

That should not mean rewriting your app, rebuilding your integration logic, or scattering provider-specific code across your stack.

Valymux gives you a single, stable way to work across providers so your team can move faster and stay sane.

## Why We Exist

AI development is becoming infrastructure work.

Teams are juggling:
- multiple providers
- changing request formats
- different tool and streaming behaviors
- inconsistent model capabilities
- secret handling across systems
- poor visibility into what is actually happening

Valymux exists to remove that pain.

## What We Are Building

- A gateway for AI traffic
- A provider abstraction layer
- A model routing and translation layer
- A secure credential vault for provider keys
- Virtual API keys for controlled access
- Observability for requests, latency, errors, and usage
- OpenAI-compatible chat completions first, with more providers expanding over time

## The Three Pillars

### Safety

We treat security and access control as core product value, not optional polish.

### Speed and Concurrency

Valymux is being built in Rust for performance, throughput, and predictable behavior under load.

### Observability

If AI infrastructure is a black box, teams lose trust fast.
We want every request to be easy to understand, trace, and improve.

## Developer Experience Promise

We are not trying to turn good engineers into platform operators.

We want the experience to feel simple:
- one integration
- one interface
- one mental model
- fewer provider headaches
- less glue code
- more time shipping product
